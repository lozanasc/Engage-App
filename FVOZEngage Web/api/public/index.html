FVOZ Engage API Documentation

What is FVOZ Engage?

Fellowship of the Voice of Zion Inc. App-Based Church Management System is a system that aids the church organization in managing, automating and organizing daily operations. It handles processes such as databases, communication, community and event and worship presentation programs, among many others. This is an app-based for the user like the district pastors and members of the church. The holder or the admin of this system which is the Fellowship of the Zion Headquarters Church will have a Web-based feature. It is free and downloadable through online, users won’t need to spend money to use it, they only need data or internet connection.

FVOZ Engage Application Architecture

Users

POST /engage/db_api/v1/user/auth
> sends email and password in JSON format as payload and returns user information in JSON format

POST /engage/db_api/v1/user/register
> sends email, full name and password in JSON format as payload and returns result in JSON format

POST /engage/db_api/v1/user/change_email
> This is a protected endpoint, sends old email and new email in JSON format as payload and returns result in JSON format.

POST /engage/db_api/v1/user/change_name
> This is a protected endpoint, sends old name and new name in JSON format as payload and returns result in JSON format.

POST /engage/db_api/v1/user/change_password
> This is a protected endpoint, sends old passsword and new password in JSON format as payload and returns result in JSON format.

Facebook

GET /engage/fb_api/v1/events/posts
> Uses facebook’s graph api to grab feed data from a particular page returns data in JSON format

GET /engage/fb_api/v1/events/events
> Uses facebook’s graph api to grab events data from a particular page returns data in JSON format

Youtube

GET /engage/yt_api/v1/sermon/sermons
> Uses ytpl to grab playlist from a public YouTube playlist in JSON format

Nodemailer

POST /engage/mail_api/v1/request/mail
> Sends subject and message in JSON format as a payload and returns the request result in JSON format

Projects

GET /engage/project/v1/projects/list
> Grabs the list of projects by name in the Database in a JSON format

POST /engage/project/v1/projects/new
> Sends name and description in JSON as payload and returns the request result in JSON format

POST /engage/project/v1/projects/edit
> Sends id, name, description, status in JSON as payload and returns the request result in JSON format

POST /engage/project/v1/projects/delete
> Sends id in JSON as payload and returns the request result in JSON format
